Export aufgezeichneter Anlagendaten

Hier können Sie die aufgezeichneten Daten als Excel-Tabelle oder SQLite-Datenbank (Rohwerte) herunterladen.

Zeitraum:

Wählen Sie den Zeitraum, aus dem die Daten entnommen werden sollen, sowie den Werteintervall, d.h. mit welchem Zeitabstand die Werte in die Tabelle eingetragen werden sollen.

Zuordnung Tabellenspalten:

Ordnen Sie hier den Spalten der Tabelle Werte zu, indem Sie Datenpunkte im Suchfeld eingeben und mit dem Button Hinzufügen in die Liste eintragen.
Der genaue Wortlaut wird während der Eingabe vorgeschlagen. Ungültige Bezeichner werden rot hinterlegt.
Die Rehenfolge der Spalten kann anschließend mit der Maus per Drag and Drop geändert werden.
Die erste Spalte 'A' enthält immer den Zeitstempel und kann nicht geändert werden.

  1. Zeitstempel
Download:
Der Excel-Export kann einen Moment in Anspruch nehmen. Die Datenpunkte werden in der oben zugewiesenen Reihenfolge in Spalten ausgegeben. Die Excel-Datei wird in Ihren Download-Ordner heruntergeladen.
Der Datenbank-Export kann einen Moment in Anspruch nehmen. Es werden alle aufgezeichneten Tags exportiert. Die SQLite-Dateien werden als ZIP in Ihren Download-Ordner heruntergeladen. SQLite.org, SQLite Browser

Konfiguration laden / speichern:

Vergeben Sie einen Konfigurationsnamen und speichern Sie die aktuelle Konfiguration. Laden Sie eine gespeicherte Konfiguration anhand des Konfigurationsnamens.
